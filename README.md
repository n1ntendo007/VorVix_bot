# VorVix_bot v3

Продвинутая панель для Telegram-ботов на Next.js + Vercel + PostgreSQL.

## Что есть в v3

- вход и регистрация по логину/паролю;
- подключение Telegram-бота по токену;
- визуальный конструктор цепочек;
- блоки: сообщение, фото, видео, файл, аудио, голосовое, GIF, задержка, условие, случайный выбор, переход, вопрос;
- inline-кнопки: ссылка, запуск другой цепочки, быстрый текстовый ответ;
- переменные: `{first_name}`, `{username}`, `{last_message}` и ответы из блока “Вопрос”;
- ожидание ответа пользователя и продолжение другой цепочки;
- webhook Telegram из панели.

## После загрузки в GitHub

1. Дождись деплоя в Vercel.
2. Открой миграцию базы:

```text
https://vor-vixbot.vercel.app/api/setup?secret=vorvix-setup-12345
```

Должно ответить:

```json
{"ok":true,"version":"vorvix_bot_v3","builder":"advanced"}
```

3. Открой сайт, зайди в аккаунт.
4. Открой бота и нажми кнопку `Webhook`.

## Важно

Для Telegram медиа можно использовать публичную ссылку `https://...` или Telegram `file_id`.
Длинные задержки на Vercel лучше не использовать. Безопасный вариант — до 20 секунд.
